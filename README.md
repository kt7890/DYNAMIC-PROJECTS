# 🌐 Personal Portfolio Website — Kamlakar Tiwari

A dynamic, glassmorphism-styled personal portfolio website built with pure HTML, CSS, and JavaScript.

## 🔗 Live Demo
[kt7890.github.io/DYNAMIC-PROJECTS](https://kt7890.github.io/DYNAMIC-PROJECTS)

## ✨ Features

- 🔭 **Live GitHub Stats** — fetches real repos, followers, stars via GitHub API
- 💬 **Working Contact Form** — sends real emails via EmailJS (no backend)
- 👁️ **Visitor Counter** — real-time visit tracking
- ⌨️ **Typing Animation** — cycling phrases in hero section
- ✨ **Interactive Particles** — 130 particles that follow mouse
- 🌀 **Custom Cursor** — lag-following ring cursor
- 📊 **Animated Skill Bars** — fill on scroll
- 📅 **Journey Timeline** — BCA → Certifications → MCA
- 🥚 **Easter Egg** — type `hire me` anywhere on the page
- 📱 **Fully Responsive** — mobile, tablet, desktop
- 🎞️ **Loading Screen** — animated intro

## 🛠️ Tech Stack

- HTML5
- CSS3 (Glassmorphism, CSS Variables, Animations)
- Vanilla JavaScript (Fetch API, Intersection Observer, Canvas)
- EmailJS (contact form)
- GitHub REST API (live stats)
- CountAPI (visitor counter)
